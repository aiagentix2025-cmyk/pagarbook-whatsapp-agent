How it works

1. Upload
Select one or more PDF, DOCX or TXT files from your computer or drag them in.
2. Chunk
Each file is split into 1 000-character chunks with 200-character overlap for context continuity.
3. Vectorize
Each chunk is embedded using OpenAI's text-embedding-ada-002 and stored in pgvector.
4. Live in bot
Instantly available. The bot semantically searches your KB on every incoming message for relevant context.
