How it works

1. Upload
Select one or more PDF, DOCX or TXT files from your computer or drag them in.
2. Chunk
Each file is split into 1 000-character chunks with 200-character overlap for context continuity.
3. Vectorize
Each chunk is embedded using OpenAI's text-embedding-ada-002 and stored in pgvector.
4. Live in bot
Instantly available. The bot semantically searches your KB on every incoming message for relevant context.

Developer Audit & Missing Features

This page tracks features and architecture alignments for the platform.

✅ 1. Inbox & Agent Takeover

  • Completed: Chat History shows live messages.
  • Completed: Ability to actually send a message from the UI to the user (Manual Reply UI).
  • Completed: Button to manually "Pause Bot" or "Resume Bot" on demand.

✅ 2. Template Management

  • Completed: A dedicated Template Builder UI to create, edit, and submit new templates directly to Meta API without logging into Facebook Business Manager.

✅ 3. Contact Management

  • Completed: A CRM-like Address Book to save contacts, add tags/segments, and import via CSV.
  • Completed: Contacts can be managed and deleted globally per tenant.

✅ 4. Analytics Dashboard

  • Completed: Basic sent/delivered/read/failed stats per campaign.
  • Completed: Global platform analytics (Total chats, delivery rate, active vs paused chats) added to Dashboard.
  • Completed: Drill-down modal view of failed messages (clicking on a campaign shows exactly which phones failed and why).

✅ 5. Agent Features

  • Completed: Human handover logic (bot automatically pauses if user types "talk to human").
  • Completed: Web search tool implementation using DuckDuckGo to allow the bot to fetch live information.
